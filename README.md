# Atividade - Abastecimento

Esse projeto foi feito a partir de um código base disponibilizado pelo professor, com o objetivo de refatorar e melhorar a aplicação.

A ideia do sistema é simples: calcular o valor total de um abastecimento com base no tipo de combustível e na quantidade de litros informada.

---

## O que foi feito

A partir do código inicial, fiz algumas melhorias para deixar o código mais organizado e funcional:

- Transformei as funções em arrow functions
- Substituí o uso de switch por um objeto para mapear os preços
- Adicionei validações (campo vazio, valores inválidos e negativos)
- Formatei o resultado para aparecer no padrão de moeda (R$)
- Ajustei o comportamento da tecla Enter para não limpar os dados
- Melhorei um pouco o visual com CSS

---

## Tecnologias usadas

- HTML
- CSS
- JavaScript

---

## Como usar

É só abrir o arquivo `index.html` no navegador, selecionar o tipo de combustível e informar a quantidade de litros.

---

## Observação

O foco da atividade não foi criar algo do zero, mas sim entender melhor como melhorar e organizar um código já existente.
